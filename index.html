HTML Cheatsheet

Basic HTML tags

Tag Name Description
<!DOCTYPE html> DOCTYPE HTML declaration Not a tag, but information to the browser what document type to expect
<html> The HTML tag HTML element that represents the root of an HTML document
<head> The head tag HTML element that represents the metadata for the document
<body> The body tag HTML element that represents the content of the document
<header> The header tag HTML element representing the introductory content of the document
<h1>,<h2>... Section headings HTML elements representing different heading levels of the document
<p> Paragraph tag HTML element containing a paragrahp
<img> Image tag HTML element containing an image
<footer> The footer tag HTML element usually representing data such as the author, contact information etc.

Table Tags

Tag Name Description
<table> Table The wrapper elements for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing table data
<tr> Table Row The table row container
<td> Table Data The table row container
<tfoot> Table Footer The set of rows containing the footer in a table
